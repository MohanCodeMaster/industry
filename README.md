# Kishore Tech Solutions - Industrial Advisory & Project Consultancy

Production-ready, highly responsive corporate website for **Kishore Tech Solutions** built on top of the Stitch Wireframe Prototype.

## Technology Stack
- **HTML5** (Semantic structure)
- **CSS3** (Custom tokens, layout spacing, responsive variables, custom scrollbar, animations)
- **Bootstrap 5.3** (Grid system, navbar container, forms validation layout)
- **Vanilla JavaScript ES6** (Sticky headers, active routes, scroll reveals, count-up animations, form simulation states)

## Folder Structure
```
project/
│
├── index.html          # Homepage
├── about.html          # About Page
├── services.html       # Services Directory
├── industries.html     # Industries Grid
├── resources.html      # Resources Download Page
├── contact.html        # Contact Page
├── consultation.html   # Request Consultation form
│
├── css/
│   ├── style.css       # Main design tokens and structural CSS
│   ├── responsive.css  # Responsive media queries overrides
│   └── animations.css  # Scroll reveal keyframe definitions
│
├── js/
│   ├── main.js         # Global script initiator
│   ├── navigation.js   # Sticky headers, active highlights and mobile toggles
│   ├── scroll.js       # Smooth scrolls, scroll-to-top FAB, and spy elements
│   ├── animation.js    # Intersection Observers, stats counting and ripples
│   └── form.js         # Bootstrap form validation and submission simulations
```

## Features
- **Mega Menu:** Dropdown detailing services categories cleanly on desktop.
- **Scroll Spy & Top Bar:** Sticky headers that change styling on scroll, with smooth scrolling across anchors.
- **Micro-Interactions:** Modern card tilt elevations, text slide-ups, custom counter increments for stats.
- **Form Verification:** Standard client-side Bootstrap form checks with full simulation status feedback.
